# Time Series Datasets for TSFMs
## Large-Scale Source Domains
- MONSTER: Monash Scalable Time Series Evaluation Repository. arXiv, 2025. [paper](https://arxiv.org/pdf/2502.15122?) [official data](https://github.com/Navidfoumani/monster)
- Time-IMM: A Dataset and Benchmark for Irregular Multimodal Multivariate Time Series. arXiv, 2025. [paper](http://arxiv.org/abs/2506.10412) [official data](https://www.kaggle.com/datasets/blacksnail789521/time-imm/data)
- CauKer: classification time series foundation models can be pretrained on synthetic data only. arXiv, 2025. [paper](https://arxiv.org/pdf/2508.02879?) [official data](None)
- TimePFN: Effective Multivariate Time Series Forecasting with Synthetic Data. AAAI, 2025.  [paper](https://ojs.aaai.org/index.php/AAAI/article/view/34288) [official data](https://github.com/egetaga/TimePFN)
